1. Sintaxe Básica (Variáveis, Loops, Condicionais)
2. Estruturas de Dados Simples
3. [X] Programação Orientada a Objetos (Classes, Herança, Polimorfismo) -> *Adicione isso quando estudar*
4. [ ] Manipulação de Exceções
5. [ ] Integração com Banco de Dados (JDBC)

### 🚀 Como Executar
Todos os exercícios são executáveis diretamente em uma IDE como IntelliJ.
